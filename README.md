# dotfilesを簡単に作成するシェルスクリプト

## 使い方

パスを通したフォルダに設置し、実行権限を追加してください。

設定例
```
mkdir ~/bin
echo '~/bin' >> ~/.bash_profile
git clone repository ~/your/path
cp ~/your/path/dotfiles ~/bin/dotfiles
chmod 755 dotfiles
```

## 設定方法

shell scriptをEditorで開き12~14行目を編集してください。

## 動作確認環境

* ubuntu 16.04 LTS
* ubuntu 18.04 LTS
